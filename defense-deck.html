Evaluation & regression

A test that passes for the right reason.

A fix that changes behavior isn't the same as a fix. So every confirmed exploit is pinned to a deterministic signal and a build.

predicate_fired
The one-sentence deterministic reason "success" fired. The regression test asserts the signal — not a generic pass.
target_version
Every verdict is attributed to a build (the platform manufactures the id) → answers "more or less resilient over time?" and catches reappearance + cross-category regressions.
